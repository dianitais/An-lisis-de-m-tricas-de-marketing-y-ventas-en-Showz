# Análisis de métricas de marketing y ventas en Showz
Este proyecto analiza datos de Showz, una empresa de venta de entradas, para optimizar gastos de marketing. Se estudiaron registros de visitas, pedidos e inversiones publicitarias para responder preguntas clave sobre adquisición y rentabilidad. 
Los objetivos incluyen:

- Evaluar el comportamiento de usuarios y su conversión a clientes.
- Calcular métricas como LTV, CAC y ROMI por canal de adquisición.
- Visualizar tendencias y evaluar el impacto de las campañas de marketing.
- Proporcionar recomendaciones basadas en datos para mejorar la rentabilidad.
